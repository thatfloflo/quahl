Quahl Browser Icon Set

The Quahl icon set comes in three styles with two variants each:

  1. Color:
    • Full colour icon: ./color/[icon-name]_c.svg
    • Greyed-out icon: ./color/[icon_name]_g.svg
    Suitable for both light and dark mode UIs. To be used primarly for window icons and primary tool bars.
  2. Light-mode Outlines:
    • Black outline icon: ./outline/[icon-name]_o.svg
    • Greyed-out outline icon: ./outline/[icon_name]_o50.svg
    Suitable only for light mode UIs. To be used in context menus, subordinate tool bars, etc. — everywhere were the full colour icons might be distracting otherwise.
  3. Dark-mode Outlines:
    • White outline icon: ./darko/[icon-name]_d.svg
    • Greyed-out outline icon: ./darko/[icon_name]_d50.svg
    Suitable only for dark mode UIs. To be used in context menus, subordinate tool bars, etc. — everywhere were the full colour icons might be distracting otherwise.

In general, the full colour / full outline variants are used for active control states (e.g. accepting input), and the greyed-out variants for inactive/off/disabled states (e.g. not accepting input).

To use the icons from Python, import Icons and/or OutlineIcons from quahl.browser.resources. Icons contains QIcon objects for the Color style, while OutlineIcons contains the Light-mode Outlines and the Dark-mode Outlines, automatically showing the appropriate style based on the user's colour scheme.

New icons can normally be added by giving just the _c, _g and o variants. There are some included scripts which attempt to automatically derive the other styles and variants from these. Where that doesn't work, they should be created manually and added as an exception to the appropriate script, with an additional *.bak.svg copy included alongside it.

This icon set is largley based on (and/or inspired by) Goran Spasojevic's Glyphs.
The full colour style is based on Glyphs's poly variant, the greyed-out varant for the color style on the duo variant (with rgba(127, 127, 127, 1.0) for stroke and rgba(200, 200, 200, 0.25) for the fill), and the outline styles and variants on the path variant (with rgba(0, 0, 0, 1.0) for the 4px stroke of the black outline, reducing stroke to 1px and setting opacity to 0.5 for the greyed-out outlines, and inverting the palette for the dark-mode outlines).
